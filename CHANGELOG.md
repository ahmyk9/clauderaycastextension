# Ask Claude Desktop Changelog

## [Initial Version] - {PR_MERGE_DATE}

- Type a prompt in Raycast, press Tab, and it opens the Claude desktop app in a new chat with your prompt submitted.
- Cross-platform: macOS and Windows.
- Uses Claude's `claude://claude.ai/new?q=…` deep link, so the prompt always lands in a regular chat (never Cowork or Code) and works even when Claude is closed.
